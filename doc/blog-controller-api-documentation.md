**请求**

- **方法**: `GET`
- **路径**: `/blog`

```json
{

  "data": [

      "createTime": "2025-12-17T10:30:00",
      "updateTime": "2025-12-17T10:30:00"
    }
```

### 2. 发布博客

用户发布新的博客文章。

**请求**

- **方法**: `POST`
- **路径**: `/blog`
- **权限**: `BLOG.ADD`
- **Content-Type**: `application/json`

**请求体**

```json
{
  "userId": 1234567890123456789,  // 可选，不传则自动从token获取
  "title": "博客标题",              // 必填，最多50字符
  "content": "博客内容",            // 必填，最多1000字符
  "tagIds": [1, 2, 3]              // 必填，至少选择一个标签
}
```

**字段说明**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | Long | 否 | 用户ID，不传则从token中自动获取 |
| title | String | 是 | 博客标题，长度不超过50字符 |
| content | String | 是 | 博客内容，长度不超过1000字符 |
| tagIds | List\<Long\> | 是 | 标签ID列表，至少选择一个 |

**响应**

- **成功响应** (200 OK)

```json
{
  "code": 200,
  "msg": "success",
  "data": null
}
```

- **失败响应**

```json
{
  "code": 500,
  "msg": "添加失败",
  "data": null
}
```

### 3. 分页条件查询博客

根据条件分页查询博客列表。

**请求**

- **方法**: `POST`
- **路径**: `/blog/query`
- **权限**: `BLOG.LIST`
- **Content-Type**: `application/json`

**请求体**

```json
{
  "userId": 1234567890123456789,      // 可选，不传则自动从token获取
  "title": "搜索关键词",               // 可选，模糊搜索，为空则查全部
  "createTime": "2025-12-17T00:00:00", // 可选，查当天数据，为空则查全部
  "pageNo": 1,                         // 必填，页码
  "pageSize": 10                       // 必填，每页大小
}
```

**字段说明**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| userId | Long | 否 | 用户ID，由后端解析并自动填充 |
| title | String | 否 | 标题模糊搜索，最多50字符，为空默认查全部 |
| createTime | LocalDateTime | 否 | 查当天数据，为空默认查全部 |
| pageNo | Integer | 是 | 页码，从1开始 |
| pageSize | Integer | 是 | 每页记录数 |

**响应**

- **成功响应** (200 OK)

```json
{
  "code": 200,
  "msg": "success",
  "data": {
    "records": [
      {
        "id": "1234567890123456789",
        "userId": "9876543210987654321",
        "title": "博客标题",
        "content": "博客内容",
        "createTime": "2025-12-17T10:30:00",
        "updateTime": "2025-12-17T10:30:00"
      }
    ],
    "total": 100,
    "pageNo": 1,
    "pageSize": 10
  }
}
```

## 通用说明

### 认证与授权

所有接口都需要用户登录并携带有效的 JWT token，通过 `Authorization` header 传递：

```
Authorization: Bearer {your-jwt-token}
```

每个接口都有对应的权限要求，用户需要拥有相应权限才能访问。

### 权限列表

| 权限代码 | 说明 |
|---------|------|
| BLOG.LIST | 查看博客列表 |
| BLOG.ADD | 添加博客 |
| BLOG.DELETE | 删除博客 |
| BLOG.EDIT | 编辑博客 |
| BLOG.STAR | 收藏/取消收藏博客 |
| COMMENT.LIST | 查看评论列表 |
| COMMENT.ADD | 添加评论 |
| COMMENT.DELETE | 删除评论 |
| USER.LIST | 查看用户信息 |

### 响应格式

所有接口均返回统一的响应格式：

```json
{
  "code": 200,        // 状态码
  "msg": "success",   // 提示信息
  "data": {}          // 响应数据，可能为对象、数组或null
}
```

### 状态码说明

| 状态码 | 说明 |
|--------|------|
| 200 | 请求成功 |
| 401 | 未授权或权限不足 |
| 404 | 资源未找到 |
| 500 | 服务器内部错误 |

### 数据类型说明

- **Long**: 长整型数字，在 JSON 中以字符串形式传输（如 `"1234567890123456789"`），避免精度丢失
- **LocalDateTime**: 日期时间格式，使用 ISO 8601 格式（如 `"2025-12-17T10:30:00"`）
- **Integer**: 整数
- **String**: 字符串
- **Boolean**: 布尔值

### 分页参数说明

分页查询接口的 PageVO 响应格式：

```json
{
  "records": [],     // 当前页数据列表
  "total": 100,      // 总记录数
  "pageNo": 1,       // 当前页码
  "pageSize": 10     // 每页大小
}
```
